# TextureCreate Studio の使い方

TextureCreate Studio は、ゲームエフェクトで使う発光、衝撃波、流線、ノイズ、閃光、SFグリッド、水面の網目模様、炎、煙、稲妻、渦巻き、火花風のテクスチャをブラウザ上で生成するツールです。

## 1. テクスチャの種類を選ぶ

左側の **Type** から作りたいテクスチャを選びます。

- **Energy Orb**: 発光する円形オーラ
- **Impact Ring**: 衝撃波や爆発リング
- **Speed Lines**: 流線や斬撃エフェクト
- **Magic Noise**: 炎、煙、魔法表現に使いやすいノイズ
- **Star Burst**: 閃光やキラメキ
- **Tech Grid**: SF UI やホログラム風グリッド
- **Water Caustics**: セル状に光が集まる水面の網目模様
- **Flame Wisps**: 炎やオーラの揺らぎ
- **Smoke Puff**: 煙や雲の柔らかい塊
- **Lightning Bolt**: 稲妻や電撃ライン
- **Magic Spiral**: 魔法陣や渦巻き
- **Spark Field**: 火花や粒子散布

プレビュー下のタイプ一覧からも切り替えできます。

## 2. 色を調整する

カラーパレットをクリックすると、雰囲気に合わせた色セットへすぐに切り替えられます。

より細かく調整したい場合は、次のカラー入力を変更します。

- **colorA**: メインカラー
- **colorB**: 外側やグラデーションに使うサブカラー
- **colorC**: 中心や強い発光部分に使うハイライトカラー

## 3. パラメーターを調整する

スライダーを動かすと、右側のプレビューが更新されます。

- **size**: 出力画像サイズ
- **seed**: ランダムパターンの固定値
- **intensity**: 効果の強さ
- **softness**: グラデーションやぼかし感
- **noise**: 粒や揺らぎの量
- **twist**: ねじれや曲がり具合
- **count**: 線、粒、放射要素などの数
- **thickness**: 線やリングの太さ
- **alpha**: 透明度

## 4. 表示オプションを切り替える

必要に応じてチェックボックスを切り替えます。

- **ramp**: 外側へ自然に消えるグラデーションを使う
- **checker**: 透明感を確認しやすいチェッカー背景を表示する
- **blackBg**: 黒背景を表示する
- **animate**: プレビューをアニメーション表示する

## 5. PNGとして保存する

プレビュー右上の **PNG 保存** をクリックすると、現在の設定で生成されたテクスチャを PNG 画像として保存できます。

保存されるファイル名には、選択中のテクスチャタイプと seed が含まれます。

## 6. プリセットを保存・読み込みする

同じ設定をあとで使いたい場合は、プリセット機能を使います。

- **Save Preset**: 現在の設定をブラウザに保存する
- **Load**: ブラウザに保存した設定を読み込む
- **JSON**: 現在の設定を JSON ファイルとして書き出す
- **Import**: 書き出した JSON ファイルを読み込む

## 7. 使い方の例

### 魔法弾の発光テクスチャ

1. **Type** を `Energy Orb` にする
2. 青系または紫系のパレットを選ぶ
3. **softness** を高めにする
4. **noise** を少し上げる
5. **PNG 保存** で書き出す

### 爆発の衝撃波リング

1. **Type** を `Impact Ring` にする
2. オレンジ系のパレットを選ぶ
3. **thickness** を上げる
4. **count** を調整する
5. **PNG 保存** で書き出す

### 斬撃・スピード線

1. **Type** を `Speed Lines` にする
2. **twist** を調整して流れを作る
3. **count** を増やして密度を上げる
4. 必要に応じて **alpha** を下げる
5. **PNG 保存** で書き出す


### 水面の網目模様

VoronoiNoise 風のセル境界を使い、黒い中心と白く光る境界線の水面模様を作ります。

1. **Type** を `Water Caustics` にする
2. 水色や白系のパレットを選ぶ
3. **count** を調整してセルの数を決める
4. **thickness** を上げて白い境界線を太くする
5. **intensity** を上げて明暗差を強める
6. **alpha** で全体の濃さを調整する
7. **PNG 保存** で書き出す


### 炎やオーラ

1. **Type** を `Flame Wisps` にする
2. 赤、オレンジ、紫系のパレットを選ぶ
3. **twist** を上げて揺らぎを強くする
4. **noise** と **thickness** で炎の太さを調整する

### 煙や雲

1. **Type** を `Smoke Puff` にする
2. 白、灰色、青系のパレットを選ぶ
3. **noise** を上げて粒の大きさを調整する
4. **alpha** を下げて半透明の煙にする

### 稲妻や火花

1. **Type** を `Lightning Bolt` または `Spark Field` にする
2. 白、青、黄色系のパレットを選ぶ
3. **count** で本数や粒数を増減する
4. **thickness** と **intensity** で発光感を調整する
